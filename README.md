# Wealthsimple-Brainstation-project
# Industry sprint project
Wealthsimple, a diverse Fintech company, has grown rapidly to a customer base of over 3 million Canadians, 
with a strategic goal to reach $1 trillion in assets within the next decade. To grow, Wealthsimple wants to 
understand how to differentiate themselves from traditional financial institutions by addressing common customer complaints.

This project for the Brainstation Data Science cohort of 3/2025 looks at that question. 

Included in this repository are 1) exec summary 2) project findings 3) notebook used for data analysis
